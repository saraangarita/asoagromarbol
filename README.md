# 🌱 ASOAGROMARBOL

Aplicación web desarrollada en React para la **Asociación de Productores Agropecuarios de Margarita, Bolívar**.

## 📌 Descripción

ASOAGROMARBOL es una asociación sin fines de lucro que promueve el desarrollo agrícola y social de la comunidad, fortaleciendo el trabajo en equipo y el compromiso con el territorio.

Este proyecto tiene como objetivo dar visibilidad a sus procesos, actividades y su impacto en la región.

## ⚙️ Instalación y uso

1. Clonar el repositorio:

```
git clone https://github.com/saraangarita/asoagromarbol.git
```

2. Entrar al proyecto:

```
cd asoagromarbol
```

3. Instalar dependencias:

```
npm install
```

4. Ejecutar el proyecto:

```
npm start
```
